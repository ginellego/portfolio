# Project Portfolio Website

Created a [portfolio web app](https://ginelle-portfolio-bmgja4f5heafcve5.canadacentral-01.azurewebsites.net/) with ASP.NET Core framework and Razor Pages. Used C#, HTML, CSS, and Javascript programming languages and Bootstrap and JQuery libraries. Deployed with Microsoft Azure as a Microsoft NETCore App. 

## Running the App Locally

Ensure dotnet is installed, then enter `dotnet run` while in the root directory. 

## Deploying to Azure

Publish the project for release. `dotnet publish -c Release`
In the Azure extension in VSCode. "Create New Web App.." under the App Services section. Then under the Web App, "Deploy to Web App".
